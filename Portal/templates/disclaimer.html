Hola!

Este es un prototipo del Portal para Afiliados desarrollado para IndustrialesMX durante mi estadía profecional (septiembre,2024 a enero,2025) requerida por mi universiadad para poder graduarme.

Por motivos legales este proyecto no muestra el producto final y funcional del verdadero Portal para Afiliados de IndustrialesMX. Tanto el aspecto visual (Front-End) como el aspecto logico (Back-End) fueron cambiados para protejer los datos privados de IndustrialesMX y sus socios.

Si bien este proyecto muestra funciones parecidas al Portal para Afiliados no tengo planeado lucrar ni plagiar la propiedad intelectual de IndustrialesMX mi unica intencion es mostrar mis conocimientos y capacidades de programacion y desarrollo aprendidas durante mi estadía profecional.
