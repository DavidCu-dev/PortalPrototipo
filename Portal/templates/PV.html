Política de Privacidad y Uso de Datos Simulados.


Este sistema web ha sido desarrollado con fines educativos, de demostración y pruebas. No está destinado a recopilar, almacenar ni procesar información personal real de individuos, empresas u organizaciones. Todas las funcionalidades han sido diseñadas exclusivamente para representar un flujo de trabajo automatizado para la generación de documentos y formularios. Se informa a los usuarios que:


  1. **Los datos ingresados deben ser completamente ficticios**, simulados o generados aleatoriamente. Está estrictamente prohibido el uso de información real, como nombres, direcciones, números de identificación, correos electrónicos, teléfonos o cualquier otro dato que pueda asociarse a personas o entidades reales.
  2. **La información almacenada en la base de datos no será utilizada, compartida ni distribuida para ningún fin**, comercial, institucional o gubernamental. Su único propósito es permitir la funcionalidad técnica del sistema.
  3. El desarrollador de este sistema no se hace responsable por el uso indebido de la plataforma o el ingreso de datos reales por parte de los usuarios. Se recomienda seguir en todo momento las buenas prácticas de desarrollo y privacidad de datos.
  4. Este sitio no utiliza cookies, rastreadores ni tecnologías de análisis. Toda la información procesada está contenida en un entorno de desarrollo o pruebas locales. Al continuar utilizando este sistema, el usuario acepta estos términos y se compromete a ingresar únicamente información ficticia.
